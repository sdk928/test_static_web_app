About This App

This is a simple static web application hosted on Azure Static Web Apps. It demonstrates how easy it is to deploy static content to Azure.

Azure Static Web Apps Features

  • Global distribution with Azure CDN
  • Custom domain support
  • Free SSL certificates
  • CI/CD integration with GitHub/Azure DevOps
  • API support via Azure Functions
